# Archiwum

Ten folder przechowuje materiały, które nie są bieżącą wersją szkolenia.

## Wersja kanoniczna

Bieżąca wersja szkolenia to 6-modułowa platforma w folderze:

- `../platforma/`

Wejście do szkolenia z katalogu głównego prowadzi przez:

- `../index.html`

## Zawartość archiwum

### `stara_wersja_4_moduly/`

Wcześniejsze materiały robocze oparte na starszej strukturze szkolenia. Zawierają dawny spis treści, research, architekturę, treści merytoryczne, ćwiczenia, promptownik oraz blueprint prezentacji.

Nie należy ich używać jako źródła bieżącej struktury kursu, bo nie są zgodne z aktualną wersją 6-modułową.

### `platforma_kopia/`

Kopia robocza wcześniejszej platformy. Została zarchiwizowana, ponieważ zawierała inną strukturę i odwołania do nieaktualnych zasobów.

### `audyty/`

Raporty audytu i materiały diagnostyczne. Mogą być używane jako historia decyzji, ale nie są materiałami do publikacji dla uczestników.
